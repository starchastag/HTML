Chapter Summary

  • All html element can have attributes.
  • The href attributes of <a> specifies the url of the page the link goes to .
  • The src attributes of <img> specifies the path to the image to be displayed
  • The width and height attributes of <img> provide size information for image.
  • The alt attribute of <img> provide an aternate text for an image .
  • The style attribute is used to add styles to an element such as color font size and more.
  • The lang attribute of the <html>tag declare the language of the Web page.
  • The title attribute define some extra information about an element.
