Defintion and Usage

The <input> tag specifies an input field an input where the user can enter data.
The <input> element is the most inportant form element.
The <input> element can be displayed in several ways depending on the type attribute.
The different input types are as follow:

  • <input type="button">
  • <input type="checkbox">
  • <input type="color">
  • <input type="date">
  • <input type="deatetime-local">
  • <input type="email">
  • <input type="hidden">
  • <input type="file">
  • <input type="image">
  • <input type="month">
  • <input type="number">
  • <input type="password">
  • <input type="radio">
  • <input type="range">
  • <input type="reset">
  • <input type="search">
  • <input type="tel">
  • <input type="text">
  • <input type="time">
  • <input type="url">
  • <input type="week">
  • <input accept="extension of file which want to you pick by the user">
