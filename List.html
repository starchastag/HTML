List in HTML

HTML offers web authors three ways for specifiying lists of information . All lists must contain one or more list elements .


  • <;ul>- An unorder list . This will list items using plain bullets.
  • <ol> - An orderd list . This will use different schemes of numbers to list your item..
  • <dl> - A definiton list . This arranges our items in the same way as they are arranged in a dictonay.

HTML Unorderd List

An unorded list is a collection of related items that have no special order or sequence . This list is created using HTML <ul> tag .Each item in the list is marked with a bullet.

Attributes of unorded list

We can ust type attributes for <ul> tag to specify the type of bullet we like .
by default it is a disc

  • <ul type = "square">
  • <ul type = "disc">
  • <ul type = "circle">

HTML Ordered List

if we are required to put our item in a numbered list instead of bulletd, then html order list will be used . This list is created by using by <ol> tag.
