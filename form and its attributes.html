HTML Form

An HTML Form is a section of a document which contains controls such as text fieds password fields checkbox , radio , buttons , submit button menus etc.



An HTML form facilities the user to enter data that is to be sent to the server for processing such as name emails address, password phone number etc.

Why use HTML Form

HTML forms are required if you want to collect some data form of the site visitor .
For example : if a user wnat to purches some items on internet he/she must fill the form such as shipping address and credit/debit details so that can be sent to the given address.
