td Tag

HTML <td> tag is used to specfic the cells of an html table which contain data of the table
The <td> tag must be child element of <tr> tag . Each table row can contain multiple <td> data elements :


The grouped<td> elements of <tr> tag renders as a single row in the table. The content of the<td>elementsis regular and left-aligned in the table by default.

Syntax


<td>.........</td>
Display Inline
Start tag/End tag Start and end tag
Usage Table content
Fixed column width using <td> tag in a table:

in an HTML table the dimensions of the rows and columns in a table are automatically adjusted by the browser so that the contents fit in the rows .
How ever there can be a situation where the width of the columns needs to be fixed or equal-sized . So there are vaious techniques to fix the width of the column.


<td< tag width attributes : The <td< tag width attributes is used to fix the width of a particular column With this we can assign a numric value to this attributes or in terms of percantage .
